Swaas AI - Frontend 🫁

This repository contains the frontend implementation of Swaas AI, a specialized web interface designed for respiratory health awareness. The focus of this project is providing a seamless, accessible, and high-performance user experience.

Tech Stack
Framework: React.js
Styling: CSS3 & Bootstrap
Icons:** Lucide-React 
Animation: Custom CSS transitions

Frontend Key Features
Dynamic Landing Page: Features an animation to guide users through the app's purpose.
Adaptive Theme System: A fully functional Light and Dark mode toggle for optimized visibility and user preference.
Responsive Design: Fully optimized for mobile, tablet, and desktop screens.
Interactive UI Components: Custom-built navigation and modular sections for symptom tracking and voice-input interfaces.

Key Files
App.js: *The main hub of the application, managing the global state (like theme toggling) and component routing.
